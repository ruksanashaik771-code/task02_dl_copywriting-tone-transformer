# Copywriting Tone Transformer

## Overview
Generates marketing copy for products using Generative AI.

## Features
- Prompt Engineering
- Groq API Integration
- Pydantic Validation
- Async Processing
- Batch Generation
- Semaphore-based Rate Limiting

## Technologies
- Python
- Groq
- Pydantic
- Asyncio
- dotenv

## Run

python -m app.main

or

python -m app.batch_main
